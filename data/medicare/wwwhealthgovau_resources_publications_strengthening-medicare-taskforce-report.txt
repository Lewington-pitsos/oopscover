# Strengthening Medicare Taskforce Report

The Strengthening Medicare Taskforce Report outlines a vision for Australia’s
primary care system of the future. The report recommends significant changes
to how primary care is funded and delivered to enable high quality, integrated
and person-centred care for all Australians.
